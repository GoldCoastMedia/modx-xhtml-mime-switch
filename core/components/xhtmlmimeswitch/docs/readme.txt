XHTML MIME Switch
=================
Version: 1.0.0-beta (2012-06-21)
Authors: Dan Gibbs <dan@goldcoastmedia.co.uk>

MODx plugin that serves polyglot markup as HTML for web browsers not compatible with XHTML/XML.

Git repo: https://github.com/GoldCoastMedia/modx-xhtml-mime-switch
